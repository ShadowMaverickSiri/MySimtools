# SimTools v2.0 - Visual Studio 2019 使用指南

## 项目结构（更新后）

```
SimTools_v2/
├── SimTools_v2.h              # 主头文件
├── SimTools_*.cpp             # 模块源文件（11个模块）
├── SimTools_test.cpp          # 测试程序（简化版）
├── CMakeLists.txt             # CMake 配置文件
├── open_vs2019.bat            # VS2019 快速启动脚本
├── build/                     # 构建目录（自动生成）
│   ├── SimTools.sln           # Visual Studio 解决方案
│   ├── SimTools_static.vcxproj # 静态库项目
│   └── SimTools_test.vcxproj  # 测试程序项目
└── Makefile                   # GCC/MinGW Makefile
```

## 编译目标

### CMake + Visual Studio 2019

**生成项目：**
```bash
# 方法1：双击运行
open_vs2019.bat

# 方法2：手动生成
mkdir build
cd build
cmake -G "Visual Studio 16 2019" -A x64 ..
```

**打开解决方案：**
- 双击 `build/SimTools.sln`
- 或运行 `open_vs2019.bat`

**在 Visual Studio 中：**
1. 选择配置：Release 或 Debug
2. 选择平台：x64
3. `SimTools_test` 已设为启动项目
4. 按 `Ctrl+Shift+B` 编译
5. 按 `F5` 或 `Ctrl+F5` 运行

### GCC/MinGW

```bash
# 编译测试程序
make

# 编译并运行
make run

# 只编译静态库
make lib

# 清理
make clean
```

## CMake 配置选项

| 选项 | 默认值 | 说明 |
|------|--------|------|
| `BUILD_TESTING` | ON | 构建测试程序 |
| `BUILD_SHARED_LIBS` | OFF | 构建动态库 |
| `USE_EIGEN` | 自动检测 | 是否使用 Eigen3 库（推荐） |
| `EIGEN_ROOT` | D:\eigen3 | Eigen3 安装路径 |

## 注意事项

### 编译配置
- 测试程序需要定义 `SIMTOOLS_STATIC` 和 `USE_EIGEN` 宏
- 如果使用 Eigen3，确保设置了正确的 `EIGEN_ROOT` 环境变量
- CMake 会自动检测并配置 Eigen3 依赖

### 控制台编码
- 测试程序已内置 UTF-8 编码设置，Windows 控制台可正确显示中文
- 使用 `SetConsoleOutputCP(65001)` 和 `SetConsoleCP(65001)` 设置编码
- 如仍有乱码，可手动在 CMD 中执行：`chcp 65001`
